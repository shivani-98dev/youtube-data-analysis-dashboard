# 📊 YouTube Engagement & Performance Dashboard

## 📌 Project Overview

This project analyzes YouTube trending video data using Tableau.
The dashboard provides insights into video performance, engagement, and trends.

---

## 📁 Dataset

* Source: Provided dataset
* Total Records: 500+ videos

---

## 📊 Key Insights

* Identified top-performing videos based on views
* Analyzed category-wise performance
* Studied engagement rate vs views
* Found trends over time

---

## 📈 Dashboard Features

* Top 10 videos analysis
* Category-wise total views
* Engagement vs views scatter plot
* Duration impact analysis
* Interactive filters

---

## 🛠 Tools Used

* Tableau Public
* Excel / CSV

---

## 🚀 How to Use

1. Download the `.twbx` file
2. Open in Tableau
3. Explore the dashboard
